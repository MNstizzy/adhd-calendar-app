# ADHD Calendar App

## Overview
The ADHD Calendar App is designed to help individuals with ADHD manage their time and tasks effectively. It features a modern and clean interface that allows users to view their tasks and calendar events in various formats, including daily, weekly, and monthly views.

## Features
- **Dashboard**: An overview of tasks and calendar events.
- **Day View**: Detailed view of tasks and events for a specific day.
- **Week View**: Weekly overview of tasks and events.
- **Month View**: Monthly calendar view for better planning.
- **Settings**: Customize user preferences for a personalized experience.
- **Focus Timer**: A built-in timer to help users focus on tasks.
- **Quick Add**: Easily add tasks without navigating away from the current view.

## Technologies Used
- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For modern and responsive styling.
- **Vite**: For fast development and build processes.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd adhd-calendar-app
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm run dev
   ```
